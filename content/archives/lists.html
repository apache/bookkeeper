Users

If you use BookKeeper or Hedwig, please subscribe to the BookKeeper user mailing list.

The BookKeeper user mailing list is: user@bookkeeper.apache.org.

  • Unsubscribe from List

Note:
In order to post to the list, it is necessary to first subscribe to it.

Developers

If you'd like to contribute to BookKeeper or Hedwig, please subscribe to the BookKeeper developer mailing list.

The BookKeeper developer mailing list is : dev@bookkeeper.apache.org.
